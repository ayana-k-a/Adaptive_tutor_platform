# 🎓 Adaptive Tutor Platform

## 🚀 Overview
Adaptive Tutor Platform is a Django-based web application that helps students connect with peer tutors based on learning style, subject, and availability.

---

## 🎯 Objective
The goal of this project is to improve peer learning by matching learners with tutors who teach in a way that suits them best.

---

## ✨ Features
- 👤 User roles: Learner & Tutor  
- 🧠 Learning style-based matching  
- 📅 Session booking system  
- 📊 Dashboard for users  
- 📝 Feedback system  

---

## 🛠️ Tech Stack
- Backend: Django  
- Frontend: HTML, CSS  
- Database: SQLite  
- Version Control: Git & GitHub

## ⚙️ Installation

1. Clone the repository:
```
git clone https://github.com/ayana-k-a/Adaptive_tutor_platform.git
```

2. Navigate to project folder:
```
cd Adaptive_tutor_platform
```

3. Create virtual environment:
```
python -m venv venv
```

4. Activate environment:
```
venv\Scripts\activate
```

5. Install dependencies:
```
pip install -r requirements.txt
```

6. Run server:
```
python manage.py runserver
```
